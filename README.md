## 📱 عن التطبيق

**نوالة** تطبيق أندرويد مخصص للسياحة الريفية في ولاية **سوق أهراس**، يتيح للمستخدمين اكتشاف الوجهات الطبيعية والتاريخية، حجز التوصيل والإقامة، وتسوّق المنتجات الريفية الأصيلة — كل ذلك من مكان واحد.

> مشروع تخرج · جامعة محمد الشريف مساعدية · سوق أهراس

